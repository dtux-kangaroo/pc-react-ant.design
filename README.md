
### 快速使用

```bash

$ git clone http://git.dtstack.cn/dtstack/dtlogux.git
$ cd dtlogux
$ yarn               # 安装依赖
$ yarn start         # 启动项目 访问http://localhost:8080
$ yarn build         # 打包

```
